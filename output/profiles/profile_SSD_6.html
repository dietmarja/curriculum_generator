Educational Profile: Sustainable Solution Designer (EQF 6)

Designs IT architectures and systems with sustainability as a core principle

Profile Overview

Role

Sustainable Solution Designer (SSD)

EQF Level

6

Main Area

Design

Thematic Area

Sustainability

Program Duration

6 semesters

Target ECTS

180 ECTS

Learning Outcomes

Graduates with this educational profile will be able to:

  • Analyze circular software design in circular economy applications
  • Develop energy-efficient software to enhance environmental performance
  • Solve lifecycle optimization in circular economy applications
  • Assess sustainable software architectures for addressing environmental challenges
  • Assess circular software design in organizational sustainability contexts
  • Examine circular software design to enhance environmental performance
  • Examine lifecycle optimization to support sustainable development goals
  • Evaluate green coding to support sustainable development goals
  • Apply integrated knowledge of circular software design, energy-efficient software to solve complex sustainability challenges

Skills

  • code refactoring for sustainability
  • green software architecture
  • energy-efficient code optimization
  • sustainable development practices
  • software carbon intensity (SCI) calculation
  • environmental impact assessment
  • stakeholder communication
  • circular economy principles application

Competencies

  • Capability in developing sustainable software lifecycle management
  • Capability in developing green code development and optimization
  • Capability in developing environmental impact assessment of software
  • Capability in developing sustainable development methodologies
  • Capability in developing technical debt management for sustainability
  • Proficiency in developing digital tools for sustainability assessment
  • Ability to analyze and interpret sustainability data for developing decision-making
  • Capacity to communicate sustainability concepts to diverse stakeholders
  • Skill in developing sustainable development frameworks in digital contexts

Assessment Methods

  • Project Work
  • Thesis
  • Case Studies
  • Reports
